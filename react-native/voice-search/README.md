## Getting Start

```Bash
# 1. Install dependencies
npm install

# 2. Run app
npx expo start
```

<br>

## Executed commands

```bash
# install react-native-voice
npm i @react-native-voice/voice --save

# Convert expo project from managed workflow to bare workflow
npx expo prebuild
```

<br>

## Local environment requirements

- Node.js >=18
- Java JDK 17
